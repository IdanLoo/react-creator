# tacer

A creator of react, using parcel-bundler, Sass, Flow and CSS Module.

## How to use

### Install globally

```sh
npm install -g create-new
create-new /path/to/project
```

### Use NPX (require npm >= 5.2.0)

```sh
npx create-new /path/to/project
```
